Instructions

  1. Make sure ReqMapper extension is installed and enabled
  2. Configure your webhook URL in the extension popup
  3. Set domain scope if needed (leave empty for all domains)
  4. Optional: Set method filter to capture only specific HTTP methods
  5. Optional: Set path regex to filter by URL path (e.g., `/api/.*` for API calls)
  6. Click the buttons above to generate requests
  7. Check your webhook endpoint for captured requests

Filter Examples:

  • Capture only API calls: Path Regex: /api/.*
  • Capture only POST requests: Method: POST
  • Ignore static files: Path Regex: ^(?!.*\.(js|css|png|jpg|gif)).*$
  • Capture specific API routes: Path Regex: ^/api/(users|posts)/
